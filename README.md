## 📌 Projeto Final 2CAW - PetCare Tracker

Projeto feito para a avaliação final de 2CAW - Criação de Aplicações WEB. O projeto aborda a construção de uma aplicação Web responsiva para registrar e acompanhar a rotina diária de cuidados de pets, como alimentação, medicação, higiene e brincadeiras.

### Como o PetTracker funciona

* O usuário vê cards individuais para cada pet com status das ações.
* Ao clicar em um botão, o sistema registra o horário atual da ação e salva os dados no navegador usando LocalStorage.
* A interface mostra quando cada tarefa foi feita, com atualização automática dos labels de tempo para exibir "Agora mesmo", "Há X min", "Hoje às" ou datas anteriores.
* Há também um reset de dados que limpa o histórico e um mecanismo de reinicialização automática diária para manter o app atualizado a cada novo dia.

### Tecnologias usadas

* HTML5 para a estrutura da página.
* Tailwind CSS para o layout, estilos e responsividade.
* Font Awesome para os ícones do pet e das ações.
* JavaScript puro para a lógica do painel, manipulação do DOM, armazenamento local e registros de ações.

### O que o projeto demonstra

* Criação de um dashboard interativo e acessível para monitorar atividades de pets.
* Uso de armazenamento local para persistência sem servidor.
* Atualização dinâmica do conteúdo com base em eventos de usuário e tempo real.
* Design simples com foco em usabilidade e feedback visual para ações do usuário.
