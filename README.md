# 🎓 Análise e Desenvolvimento de Sistemas (ADS) - Atividades Acadêmicas

> "Construindo a base para me tornar um Desenvolvedor Fullstack."

Bem-vindo ao meu repositório acadêmico! Este espaço é dedicado a armazenar, organizar e versionar todas as atividades, trabalhos e projetos práticos desenvolvidos durante a minha graduação em Análise e Desenvolvimento de Sistemas.

O objetivo aqui é documentar a minha evolução técnica ao longo dos semestres, servindo tanto como um portfólio de aprendizado quanto como um material de consulta rápida para conceitos de computação e engenharia de software.

## 🚀 Tecnologias e Ferramentas Abordadas

Durante o curso, aplico e aprofundo conhecimentos em diversas frentes do desenvolvimento. Algumas das principais tecnologias e conceitos presentes nestes projetos incluem:

* **Fundamentos e Back-end:** C (Gestão de memória, Estrutura de Dados), Node.js.
* **Front-end e UI/UX:** React, Tailwind CSS, Bootstrap, Figma.
* **Fundamentos da Computação:** Sistemas Operacionais (Escalonamento, Gerenciamento de Processos).
